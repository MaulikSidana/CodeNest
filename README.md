# CodeNest – Online Coding Practice Platform

CodeNest is a coding practice platform inspired by LeetCode where users can solve programming problems, improve algorithmic thinking, and prepare for technical interviews.

The platform allows users to browse problems, write code, test solutions, and track their progress.

---

## 🚀 Features

* Practice coding problems similar to LeetCode
* Multiple problem categories (Arrays, Graphs, DP, etc.)
* Code editor for writing solutions
* Test case execution and validation
* Problem difficulty levels (Easy / Medium / Hard)
* User progress tracking
* Clean and simple UI for problem solving

---

## 🧠 Purpose

The goal of this project is to help students and developers:

* Practice data structures and algorithms
* Prepare for coding interviews
* Improve problem solving skills
* Learn competitive programming concepts

---

## 🛠 Tech Stack

Depending on the implementation, the platform may include:

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js / Express.js 
* **Database:** MongoDB 
* **Code Execution Engine:** Docker 

---

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/codenest.git
```

2. Move into the project directory

```
cd codenest
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm start
```
